# Emotion_Detection_Using_AI

# 😊 Face Emotion Detection + AI Mood Booster

A Python-based web application that detects human emotions from face photos or webcam using Deep Learning and gives personalized mood-boosting suggestions.

## ✨ Features

- Detects **7 emotions**: Angry, Disgust, Fear, Happy, Sad, Surprise, Neutral
- Works with **Photo Upload** and **Live Webcam**
- Smart **AI-powered mood suggestions** for negative emotions (Sad, Fear, Angry, Disgust)
- Built with **TensorFlow**, **OpenCV**, and **Streamlit**
- Fully deployed online for free

## 🛠️ Technologies Used

- Python
- TensorFlow / Keras (CNN Model trained on FER-2013)
- OpenCV (Face Detection)
- Streamlit (Web Interface)
- Streamlit Cloud (Deployment)
